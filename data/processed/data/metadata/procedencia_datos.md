# Procedencia de los datos

Fuente: Yahoo Finance  
Acceso programático: API mediante librería yfinance en Python  
Conjunto descargado: Precios diarios (Close) de BTC-USD, ETH-USD, LTC-USD, DOGE-USD  
Periodo: 1 enero 2018 – 30 diciembre 2020  
Frecuencia: diaria  
Moneda: USD  

Los datos son de acceso libre bajo las condiciones de uso de Yahoo Finance.
