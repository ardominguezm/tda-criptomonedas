# Código del proyecto

Esta carpeta contiene todos los notebooks y scripts utilizados en la investigación.

- `01_descarga_datos.ipynb`: descarga los datos desde Yahoo Finance.
- `02_preprocesamiento.ipynb`: cálculo de retornos, alineamiento de series y limpieza.
- `03_pipeline_TDA.ipynb`: cálculo de diagramas de persistencia y paisajes
  utilizando Ripser y Persim.
- `04_visualizaciones.ipynb`: generación de gráficas de normas L1 y L2.
- `utils.py`: funciones auxiliares para el pipeline.

Todos los notebooks están preparados para ejecutarse en Google Colab.
